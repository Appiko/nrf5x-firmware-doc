Detailed Description

This module contains the API to manage the NUCLEO motherboard.

The supported demoboard is NUCLEO-L152RE motherboard:

  • STM32L152RE 64-pin microcontroller
  • Mini USB connector for power supply and I/O
  • ST-Link, JTAG/SWD on board debugger
  • User button and RESET button
  • 1 User LED
Note
The pin and specific struct details are located in the file stm32lxxx_nucleo.h .
